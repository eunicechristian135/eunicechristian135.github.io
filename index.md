## ❓ Troubleshooting

If you encounter issues, follow these common fixes:

*   Issue: The tool does not open.
    Fix: Ensure you have the latest .NET Runtime installed from the official Microsoft website.
*   Issue: Visuals do not appear on screen.
    Fix: Set your game to "Borderless Windowed" mode. This allows the overlay to draw on top of the game window.
*   Issue: Settings reset after closing.
    Fix: Right-click the application icon and select "Run as administrator." This gives the program permission to save configuration files in the root folder.
*   Issue: Low frame rate.
    Fix: Disable heavy background applications like video recording software or browsers to free up CPU power.
